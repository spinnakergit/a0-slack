Slack Settings

Bot Account (Primary)
From api.slack.com/apps. Required for sending messages. Required for Socket Mode (chat bridge). Enable Socket Mode in your Slack app settings.
User Token (Optional)
User tokens enable search functionality. Only install as a user if needed.

Chat Bridge
Route Slack messages through Agent Zero's LLM via Socket Mode. Requires bot token + app token. Channels are managed via the slack_chat tool.
User Allowlist
Only these Slack users can interact with the bot. Leave empty to allow all workspace members.
Elevated Mode (2FA)
When enabled, authenticated users can unlock full Agent Zero access from Slack using !auth <key>. Default mode is restricted (chat only, no tools).
SECURITY NOTICE
Elevated mode grants Slack users full access to Agent Zero's capabilities, including code execution, file operations, and system tools.

Recommended: A private workspace with only trusted members, a defined User Allowlist above, and short session timeouts.
Share this key securely. Users type !auth <key> in Slack to elevate.
Security warning: "Never expire" means an elevated session stays active until the bot restarts or the user runs !deauth.
